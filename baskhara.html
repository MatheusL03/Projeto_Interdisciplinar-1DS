Calculadora de Equação de Segundo Grau
A equação de 2 grau é uma equação que começa com um coeficiente,tem o valor X ao quadrado e é terminada com um número. Por exemplo: 3X² + 5X - 2 = 0. Pode parecer intimidador, mas não se preocupe! É por isso que estamos aqui.
Agora iremos começar ! Mas antes vamos para algumas regras. Se algum coeficiente for decimal, utilize ponto no lugar de vírgula. Por exemplo, se algum coeficiente for igual a 2,55 então digite 2.55. O coeficiente a não pode ser zero. Se a equação for uma equação incompleta, isto é, algum termo estiver ausente, substitua o coeficiente do termo ausente por zero. Exemplos:

  1. Na equação x2 + 4 = 0, os coeficientes são: a = 1, b = 0 e c = 4.

  1. Na equação -x2 + 3x = 0, os coeficientes são: a = -1, b = 3 e c = 0.
